# Core Features

LOD Generator Pro isn't just a simple polygon decimation script. It's a complete suite of optimization tools designed to fit perfectly into your Unity pipeline. 

From automated mesh generation to intelligent LODGroup configuration, these tools save you dozens of hours so you can focus on making your game fun, not manually tweaking triangles.
